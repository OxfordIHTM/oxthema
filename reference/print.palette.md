# Print a palette

Print a palette

## Usage

``` r
# S3 method for class 'palette'
print(x, ...)
```

## Arguments

- x:

  A character value or a vector of values of class palette.

- ...:

  Other arguments

## Value

Shows image of a palette with the specified colours.
